Project made for WPF classes. Uses LiveChart library for drawing expenses charts and PDFCreator for creating expenses summary in pdf for each month. Draws chart with 3 last months when window height or width is less than 440, draws chart with 6 last months when windows height or width is less than 800 and greater than 400 and draws chart with 12 last months when windows height and width are bigger than 800. The project also uses commands concept with for example.

To use this project you will need a database named "projektwpf" with string comparison method "utf8_general_ci". It works with XAMPP MySQL server. To recreate database uncomment line "db.RecreateDatabaseWithValues();" for the first time usage. You will also need something like "mysql-connector-odbc-5.1.8-win32.msi" with can be obtained from these links https://dev.mysql.com/downloads/file/?id=396600 or https://dev.mysql.com/downloads/connector/odbc/5.1.html . After installation it may be necessary to change driver name in the class "DBConnectionODBC.cs".
